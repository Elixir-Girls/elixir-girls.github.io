Additional Tools

Depending on where you are at in your programming journey here are some additional tools you may want to consider installing.

Text Editors

If you are just starting out programming you will want to install a text editor so that you easily write code.

Mac

  • Sublime Text
  • Atom

Windows

  • Sublime Text
  • Atom

Git GUI

If you are not familiar with git or are a visual learner you may find a graphical user interface a helpful tool for understanding Git Workflows.

Mac

  • Source Tree
  • Tower

Windows

  • Source Tree
  • Tower

Heroku Command Line Tool

Once you've built your app, you'll be able to deploy it to heroku with this tool

Mac

  • Heroku CLI

Windows

  • Heroku CLI
